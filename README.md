# 💸 Smart Expense Splitter – UI/UX Design

A modern, intuitive app interface designed to help users track, split, and manage shared expenses efficiently.  
The design focuses on *simplicity, clarity, and a premium dark mode aesthetic.*

---

## 🌟 Overview

**Smart Expense Splitter** makes group expense management effortless.  
Users can:
- Track incomes, spending, and savings progress  
- Split bills among participants  
- View categorized spending history and analytics  
- Set and monitor saving goals  

---

## 🎨 Tools & Technologies
- **Figma** — UI Design & Prototyping  
- **FigJam** — User flow & wireframes  
- **Google Fonts**  
- **Material Design System**
